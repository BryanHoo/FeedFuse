# 添加常用快捷键

## Goal
为阅读器中的常用操作补充统一快捷键，优先支持打开全局搜索，并兼容 macOS 与 Windows/Linux 常见组合键习惯。

## Requirements
- 为全局搜索增加 `Cmd+F` / `Ctrl+F` 快捷键支持。
- 快捷键仅在阅读器前端上下文内生效，不影响浏览器默认行为之外的其他页面。
- 当焦点位于可编辑输入区域时，不应错误拦截用户正常输入。
- 保持现有搜索弹窗行为与搜索请求逻辑不变，仅补充快捷键入口。
- 为新增快捷键行为补充或更新前端测试。

## Acceptance Criteria
- [ ] 在阅读器页面按下 `Cmd+F` 或 `Ctrl+F` 时会打开全局搜索弹窗。
- [ ] 在输入框、文本域或可编辑区域中按下相同组合键时，不会错误触发阅读器快捷键。
- [ ] 现有全局搜索功能与搜索结果交互保持正常。
- [ ] 相关单元测试通过。

## Technical Notes
- 变更范围预计集中在阅读器布局层与全局搜索相关测试。
- 键盘监听逻辑需要做好事件清理，符合前端 hook/effect 规范。
