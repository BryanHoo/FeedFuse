# Superwork Instructions

These instructions are for AI assistants working in this repository.

Only activate the superwork workflow when the user explicitly requests `use superwork`.

When active:
- PRDs live in `docs/superwork/prd/`
- Implementation plans live in `docs/superwork/plans/`
- Repository specs live in `docs/superwork/specs/`
- Read both `AGENTS.md` and `CLAUDE.md` when they exist; do not treat them as mutually exclusive
- Use `superwork-project-bootstrap` when repo-level workflow files are missing or stale
- Use `superwork-preparing-specs` before touching code when repo-specific specs are needed

Keep this managed block so future bootstrap runs can refresh it safely.
